StudentApp

This is an app to simulate a Simplified Student Management System.

Made by Thiago Jacinto, for learning proposes.

List

Returns all students inserted in the system.

Find

Returns the student that matches the following ID.

Insert

Insert a new student into the system.

Update

Updates information about some already inserted student.

Remove

Removes from the system the student that matches the following ID.
